# open-convert 
                                                                                                                                                
  open-convert 是一个多格式文件转换平台，当前实现了文档转换子模块 doc-convert/，预留了 audio-convert/ 和 video-convert/。                       
                                                                                                                                                
  ---                                                                                                                                           
  部署架构（三层）
                                                                                                                                                
  用户浏览器                                                                                                                                    
      ↓                                                                                                                                         
  Cloudflare Pages（前端 SPA）                                                                                                                  
  covert.ljhztq.com/doc-convert/                                                                                                                
      ↓ /doc-convert/api/* 请求                                                                                                                 
  Cloudflare Worker（API 网关）                                                                                                                 
  - CORS 处理 / 反代 / 路由                                                                                                                     
      ↓                                                                                                                                         
  Render.com（Python FastAPI 后端）                                                                                                             
  - 文件接收 / 转换调度 / LibreOffice                                                                                                           
                                                                                                                                                
  ┌──────────┬───────────────────┬─────────────────────┐                                                                                        
  │   层级   │       平台        │        费用         │                                                                                        
  ├──────────┼───────────────────┼─────────────────────┤
  │ 前端     │ Cloudflare Pages  │ 免费                │
  ├──────────┼───────────────────┼─────────────────────┤
  │ API 网关 │ Cloudflare Worker │ 免费（10万请求/天） │                                                                                        
  ├──────────┼───────────────────┼─────────────────────┤                                                                                        
  │ 后端     │ Render.com 免费层 │ 免费                │                                                                                        
  └──────────┴───────────────────┴─────────────────────┘                                                                                        
                  
  ---                                                                                                                                           
  技术栈          
        
  前端： Vite + Vanilla JS（无框架，bundle ~20KB），原生 CSS
                                                                                                                                                
  后端： Python 3.11 + FastAPI，Docker 部署（debian:bullseye-slim + LibreOffice + Tesseract）                                                   
                                                                                                                                                
  任务管理： FastAPI BackgroundTasks + SQLite（/tmp/docconv/jobs.db）持久化状态，asyncio.Semaphore(5) 限制并发

  转换能力矩阵    
                                                                                                                                                
  ┌───────────────┬──────────────────────────────────────────────────┬──────┐
  │     来源      │                       目标                       │ 方案 │                                                                   
  ├───────────────┼──────────────────────────────────────────────────┼──────┤
  │ PDF → Word    │ pdf2docx                                         │      │                                                                   
  ├───────────────┼──────────────────────────────────────────────────┼──────┤
  │ PDF → PPT     │ PyMuPDF（每页渲图，文字不可编辑）                │      │                                                                   
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ PDF → Image   │ PyMuPDF，多页输出 ZIP                            │      │                                                                   
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ Word → PDF    │ LibreOffice headless                             │      │
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ PPT → PDF     │ LibreOffice headless                             │      │
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ Image → PDF   │ img2pdf，支持多图有序合并                        │      │
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ Image → Word  │ Tesseract OCR（中英文）                          │      │
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ Word/PPT 互转 │ python-docx + python-pptx                        │      │
  ├───────────────┼──────────────────────────────────────────────────┼──────┤                                                                   
  │ HEIC          │ 前端 heic-convert 转 JPEG，后端 pillow-heif 兜底 │      │
  └───────────────┴──────────────────────────────────────────────────┴──────┘
